# Əzimov Xətai — Front-End Portfolio

Kiçik şəxsi portfolio vebsaytı. İçində əsas fayllar:
- `index.html`
- `stayle.css`
- `img/` (şəkillər)

How to run locally:
1. Open `index.html` in your browser, or serve with a static server:

```powershell
# Simple static server using Python 3
python -m http.server 3000
# then open http://localhost:3000
```

To push to GitHub:
1. Create a GitHub repo.
2. Run locally:

```powershell
git remote add origin https://github.com/<your-username>/<repo>.git
git branch -M main
git push -u origin main
```
